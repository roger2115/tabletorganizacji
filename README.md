# 🚀 TABLET ORGANIZACJI - Cyberpunk RP Interface

Zaawansowany interfejs tabletu w stylu Cyberpunk 2077 stworzony dla Discord RP. Pełnowymiarowa aplikacja webowa z systemem logowania, zarządzaniem danymi i mini-grami.

## ✨ Funkcje

### 🔐 System Logowania
- **Administrator** (`DrS` / `admin123`) - pełny dostęp do edycji
- **Użytkownik** (`user` / `user123`) - tryb tylko do odczytu
- Zarządzanie kontami użytkowników

### 📊 Zakładki Systemu
- **INFORMACJE** - Wiadomości systemowe i karty informacyjne
- **DANE MEDYCZNE** - Rejestr pacjentów z monitoringiem bio
- **PLIKI** - System plików z ochroną hasłem
- **PROJEKT XYZ** - Tajny projekt z paskami postępu
- **UŻYTKOWNICY** - Zarządzanie kontami (tylko admin)
- **ZARZĄDZANIE** - Panel konfiguracji systemu

### 🎮 Mini-Gry
- **Aimlab** - Trening celności w stylu cyberpunk
- Dostępne z paska zadań

### 🎵 Audio & Efekty
- Oryginalne dźwięki z Cyberpunk 2077 (embedded base64)
- Muzyka w tle z YouTube
- Efekty atmosferyczne (scanlines, glitch, migotanie)

### 💾 Trwałość Danych
- Wszystkie dane zapisywane w localStorage
- Prefiks `rp_` dla wszystkich kluczy
- Automatyczne ładowanie przy starcie

## 🛠️ Instalacja

1. Pobierz `index.html`
2. Otwórz w przeglądarce
3. Gotowe! Nie wymaga serwera.

## 🎨 Stylizacja

- **Paleta kolorów**: Czerwony `#e8003d`, Cyan `#00e5cc`, Żółty `#ffd200`
- **Fonty**: Share Tech Mono, Rajdhani
- **Motyw**: Cyberpunk 2077 inspired
- **Responsywność**: Pełny viewport, tablet-like interface

## 🔧 Konfiguracja

Panel zarządzania pozwala na:
- Regulację głośności efektów i muzyki
- Ustawienia wizualne (scanlines, glitch)
- Edycję konfiguracji JSON
- Reset danych

## 📱 Pasek Zadań

Realistyczne funkcje tabletu:
- WiFi/Bluetooth toggle
- Regulacja jasności
- Powiadomienia
- Informacje o baterii i sieci
- Zegar z kalendarzem
- Mini-gra Aimlab

## 🚀 Użycie

1. **Boot sequence** - Automatyczne uruchomienie z efektami
2. **Logowanie** - Wybierz rolę i wprowadź dane
3. **Nawigacja** - Klikaj zakładki w górnym pasku
4. **Edycja** - Przycisk "EDYTUJ" dla administratorów
5. **Mini-gry** - Przycisk "AIMLAB" w pasku zadań
