# 🎮 TABLET ORGANIZACJI

Cyberpunk-styled tablet interface dla Discord RP z pełnym systemem zarządzania.

![Version](https://img.shields.io/badge/version-5.0-red)
![License](https://img.shields.io/badge/license-MIT-cyan)

## 🚀 Demo

Strona jest automatycznie wdrażana na GitHub Pages po każdym pushu do brancha `main`.

## ✨ Funkcje

- 🔐 System logowania z rolami (admin/user)
- 📋 Zarządzanie wiadomościami systemowymi
- 🏥 Rejestr medyczny z body scan 3D
- ⚗️ Projekt XYZ - dane niejawne
- 👥 Zarządzanie użytkownikami (admin)
- ⚙️ Panel konfiguracji z suwakami
- 🎵 Dźwięki Cyberpunk 2077
- 🎨 Pełna personalizacja (kolory, audio, wizualizacja)

## 🎨 Technologie

- HTML5 + CSS3 + Vanilla JavaScript
- Embedded Cyberpunk 2077 UI SFX
- localStorage dla persystencji danych
- SVG dla body scan wizualizacji
- Google Fonts (Share Tech Mono, Rajdhani)

## 📦 Struktura

```
Strona/
├── index.html          # Główna aplikacja (self-contained)
├── README.md           # Dokumentacja
└── .github/
    └── workflows/
        └── deploy.yml  # GitHub Actions deployment
```

## 🛠️ Rozwój

Plik `index.html` jest self-contained - zawiera wszystko (HTML, CSS, JS, audio jako base64).
